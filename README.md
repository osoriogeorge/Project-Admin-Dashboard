# Project-Admin-Dashboard

This project is a sample dashboard built with HTML, CSS Grid, and Flexbox.

## Description

The dashboard provides an interface for visualizing and managing data. It includes sections for:

- **Header**: Info user,search, and user actions.
- **Sidebar**: Contains the logo, Main navigation for the dashboard.
- **Main content**: Displays relevant information and widgets.
- **Footer**: Copyright.

## Technologies used

- HTML5
- CSS3 (Grid and Flexbox)

## How to run the project

1. Clone this repository: `git clone https://github.com/osoriogeorge/Project-Admin-Dashboard.git`
2. Open the `index.html` file in your browser.

## Project structure

dashboard-project/
├── index.html
├── styles.css
└── README.md

## Design

The dashboard design is based on CSS Grid for the main layout and Flexbox for aligning and distributing elements within each section.

- **Grid**: Defines the overall structure of the dashboard in rows and columns.
- **Flexbox**: Aligns and distributes elements within containers such as the header, sidebar, and main content.
